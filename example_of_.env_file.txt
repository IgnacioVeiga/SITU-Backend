Use this file as example to create your own ".env" file.

DB_URL=jdbc:postgresql://localhost:5432/situ
DB_USERNAME=admin
DB_PASSWORD=admin@1234

MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=example@gmail.com
MAIL_PASSWORD=example-password

Must be a long base64 string without the "=", for example:
JWT_SECRET=MUwtLVAtKjZJRVwjJT0jZj5lcDxIanY6R1BjPGEyP1V2eSxiXmErbWQhc2NRKHdK
